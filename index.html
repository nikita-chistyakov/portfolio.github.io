Few-shot learning technique

Création d'une application web orientée client permettant aux utilisateurs de télécharger une image d'interrogation et un ensemble de soutien composé de seulement trois images. Ce programme utilise un algorithme d'apprentissage avancé pour analyser l'image de la requête et identifier l'image la plus similaire à partir de l'ensemble de support. Le résultat est ensuite présenté dans une interface de navigateur web conviviale. Ce projet, fruit d'une collaboration entre trois membres de l'équipe, intègre de manière transparente des techniques innovantes d'apprentissage automatique et une expérience utilisateur intuitive.

Built a client-facing web application enabling users to upload a query image and a support set of only three images.
This program employs advanced few-shot learning algorithm to analyze the query image and identify the most similar image from the support set. The result is then presented within a user-friendly web browser interface.
This project, a collaborative effort by a three-member team, seamlessly integrates innovative machine learning techniques with an intuitive user experience.
